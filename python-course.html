⚙️ Cài đặt Python & Môi trường
Hướng dẫn từng bước để cài đặt Python và chạy code đầu tiên của bạn.
1
Tải & Cài đặt Python
Truy cập python.org/downloads và tải phiên bản mới nhất (Python 3.12+). Luôn chọn "Add Python to PATH" khi cài trên Windows.
CMD / PowerShell
# Kiểm tra Python đã cài chưa
python --version

# Nếu chưa thấy → tải tại python.org/downloads
# ✅ Nhớ tick "Add Python to PATH"
Terminal
# Dùng Homebrew (khuyên dùng)
brew install python

# Hoặc tải .pkg tại python.org
python3 --version
Terminal
# Ubuntu / Debian
sudo apt update && sudo apt install python3

# Kiểm tra phiên bản
python3 --version
2
Cài VS Code (Trình soạn thảo code)
Tải VS Code tại code.visualstudio.com — miễn phí, nhẹ, và cực kỳ phổ biến. Sau khi cài, mở VS Code và cài extension Python (của Microsoft).
💡
Ngoài VS Code, bạn cũng có thể dùng PyCharm (mạnh hơn, nặng hơn) hoặc Jupyter Notebook (tốt cho data science).
3
Chạy chương trình Python đầu tiên
Tạo file hello.py và gõ đoạn code sau:
hello.py
# Chương trình Python đầu tiên của bạn!
print("Xin chào, Python! 🐍")
print("Tôi đang học lập trình!")
▶ Output
Xin chào, Python! 🐍
Tôi đang học lập trình!
🎉
Nếu bạn thấy output trên — chúc mừng! Môi trường của bạn đã sẵn sàng!
